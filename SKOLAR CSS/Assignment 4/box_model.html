CSS Box Model

This page demonstrates the CSS box model with examples showing the effect of padding, border, and margin on different elements.

content-box model

border-box model

Explanation

In the content-box model, the width and height you set for an element only apply to the content. Padding, border, and margin are added outside the content dimensions.

In the border-box model, the width and height you set include the content, padding, and border. The total size of the element does not increase with padding and border.
